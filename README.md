サポート系の機能を追加します  



## 機能一覧
- Cancel
- BlockClick
- EntityClick
- Text



## Cancel
ブロックをクリックした際のイベントを無効化します  
特定のタグを所持していた場合、無効化をスルーすることも可能です  
> 例: チェスト、ドアをロックなど  

### 1. 登録
無効化したいブロックの座標を入力することで登録します  
コマンド: `/blick.bind 10 50 10`   
(座標 x10, y50, z10 にあるブロックがクリックされた際のイベントを無効かします)
